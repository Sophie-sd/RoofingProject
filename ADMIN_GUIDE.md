## 3. Загальні правила (запамʼятайте)

### Збереження змін

Після будь-яких правок натисніть одну з кнопок унизу форми:

```
┌──────────────────────────────────────────────────────────┐
│                                                          │
│   [ Зберегти ]   [ Зберегти і продовжити ]               │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

| Кнопка | Коли натискати |
|--------|----------------|
| **Зберегти** | Зберегти і повернутися до списку |
| **Зберегти і продовжити редагування** | Зберегти і залишитися на цій же формі |

Без натискання кнопки зміни **не потраплять** на сайт.

### Прапорці видимості

| Поле | Що робить |
|------|-----------|
| **Активне** / **Показувати на головній** | Якщо зняти галочку — елемент зникне з сайту (або з головної) |
| **На головній** (`show_on_home`) | Показувати картку/фото також на головній сторінці |

### Порядок (`Порядок`)

Чим **менше** число — тим **вище** елемент у списку (0, 1, 2…).

У багатьох списках можна змінити порядок прямо в таблиці (колонка «Порядок»), потім натиснути **Зберегти** унизу списку.

### Зображення

Зазвичай є два способи:

| Поле | Як користуватися |
|------|------------------|
| **Зображення / Фото (upload)** | Натисніть «Обрати файл» і завантажте з компʼютера (краще JPG/WebP до ~2 МБ) |
| **URL зображення** | Залиште порожнім, якщо вже завантажили файл. Це запасний варіант |

Якщо є і файл, і URL — на сайті показується **завантажений файл**.

### Текстовий редактор (TinyMCE)

У полях на кшталт «Текст сторінки» / «Відповідь» FAQ є панель форматування (жирний, списки, посилання). Працює як у Word: виділіть текст → натисніть потрібну кнопку.

## 4. Налаштування сайту (телефон, бренд, SEO)

**Меню:** Налаштування → **Налаштування сайту**

Відкривається одразу форма (окремого «списку» немає).

```
┌─ Налаштування сайту ─────────────────────────────────────┐
│                                                          │
│  Бренд                                                   │
│    Назва сайту / рядок 1 / рядок 2                       │
│                                                          │
│  Контакти                                                │
│    Телефон · Email · Локація                             │
│                                                          │
│  Години роботи                                           │
│    Будні · Вихідні · Підказка форми                      │
│                                                          │
│  Статистика                                              │
│    Проекти · Досвід · Гарантія  (цифри на банері)        │
│                                                          │
│  Бренди / SEO                                            │
│    Бренди матеріалів (через кому) · Опис для Google      │
│                                                          │
│  Соцмережі                                               │
│    Facebook · Instagram · YouTube                        │
│                                                          │
│              [ Зберегти ]                                │
└──────────────────────────────────────────────────────────┘
```

| Що змінити | Де на сайті зʼявиться |
|------------|------------------------|
| Телефон, email, локація | Шапка, підвал, сторінка контактів, кнопки «подзвонити» |
| Години роботи | Контакти / форми |
| Статистика (400+, 20+, 10) | Цифри на головному банері |
| Бренди матеріалів | Текст «про нас», якщо там є плейсхолдер брендів |
| SEO-опис | Мета-опис для пошуковиків |
| Соцмережі | Посилання у футері (якщо виведені на сайті) |

**Важливо:** змінений **телефон** автоматично оновлює посилання для дзвінка (`tel:`). Достатньо вписати номер у зручному вигляді, наприклад `096 409 66 12`.

## 5. Головна сторінка (`https://pokrivlya.com.ua/`)

Головна складається з кількох блоків + стрічки + карток послуг/портфоліо з інших розділів.

### 5.1. Блоки головної

**Меню:** Головна → **Блоки головної**

```
┌─ Блоки головної ─────────────────────────────────────────┐
│                                                          │
│  Назва в адмінці          │ Показувати │ Заголовок       │
│  ─────────────────────────┼────────────┼───────────────  │
│  Головний банер           │ ☑          │ …               │
│  Про нас (превʼю)         │ ☑          │ …               │
│  Послуги (превʼю)         │ ☑          │ …               │
│  Портфоліо (превʼю)       │ ☑          │ …               │
│  Форма зворотного звʼязку │ ☑          │ …               │
│                                                          │
│  → Клікніть на назву блоку, щоб відкрити редагування     │
└──────────────────────────────────────────────────────────┘
```

Нові блоки **додавати / видаляти не можна** — лише редагувати існуючі.

---

#### А) Головний банер

**Відкрити:** Блоки головної → **Головний банер**

```
┌─ Головний банер ─────────────────────────────────────────┐
│  ☑ Показувати на головній                                │
│                                                          │
│  Заголовок                                               │
│    Рядок 1 ……  (напр. НАДІЙНА)                           │
│    Рядок 2 ……  (напр. ПОКРІВЛЯ.)                         │
│    Акцентний рядок …… (напр. ПІД КЛЮЧ.)                  │
│    Лід / опис …… короткий текст під заголовком           │
│                                                          │
│  Кнопки                                                  │
│    Кнопка 1 — текст / посилання                          │
│    Кнопка 2 — текст / посилання                          │
│                                                          │
│  Підписи статистики                                      │
│    Стат 1 / 2 / 3 — підписи під цифрами                  │
│    (самі цифри — у «Налаштування сайту» → Статистика)    │
│                                                          │
│  Фон                                                     │
│    Зображення (файл) · Превʼю · URL (запасний)           │
└──────────────────────────────────────────────────────────┘
```

| Хочу змінити | Що редагувати |
|--------------|---------------|
| Великий заголовок | Рядки 1, 2, акцент |
| Текст під заголовком | Лід / опис |
| Кнопки | Текст і посилання кнопок |
| Фон банера | Зображення |
| Цифри 400+ / 20+ / 10 | **Налаштування сайту** → Статистика |
| Підписи під цифрами | Стат 1–3 у цьому блоці |

Посилання кнопок часто виглядають так: `#contact`, `#portfolio` (прокрутка на секцію) або `/services/` (інша сторінка).

---

#### Б) Про нас (превʼю на головній)

**Відкрити:** Блоки головної → **Про нас (превʼю)**

| Поле | Що це |
|------|--------|
| Підпис над заголовком | Маленький текст зверху |
| Заголовок | Основний заголовок секції |
| Текст 1 / Текст 2 | Абзаци |
| Чеклист | Список переваг — **кожен пункт з нового рядка** |
| Текст / URL посилання | Кнопка «Детальніше…» |
| Фото | Зображення секції |

---

#### В) Послуги (превʼю) і Портфоліо (превʼю)

Тут редагуються лише **заголовки секцій** і посилання «Всі послуги» / «Усі проекти».

Самі **картки послуг** і **фото портфоліо** беруться з розділів:

| Що на головній | Де редагувати вміст |
|----------------|---------------------|
| Картки послуг | Контент → **Послуги (картки)** + галочка **На головній** |
| Фото робіт | Контент → **Портфоліо** + галочка **На головній** |

---

#### Г) Форма зворотного звʼязку (CTA)

**Відкрити:** Блоки головної → **Форма зворотного звʼязку**

| Поле | Що змінює |
|------|-----------|
| Підпис / Заголовок / Лід | Тексти біля форми |
| Заголовок форми | Заголовок самої форми |
| Підказка форми | Текст на кшталт «Відповімо протягом дня…» |

Поля форми (населений пункт, телефон тощо) **не змінюються** в адмінці — це частина сайту.

### 5.2. Стрічка на головній

**Меню:** Головна → **Стрічка на головній**

|-----|----|

## 7. Типовий сценарій роботи

```
1. Увійдіть → https://pokrivlya.com.ua/admin/
2. Ліворуч оберіть потрібний розділ (див. шпаргалку вище)
3. Відкрийте запис (клік по назві) або натисніть «Додати…»
4. Змініть поля
5. Натисніть [ Зберегти ]
6. Відкрийте відповідну сторінку сайту і перевірте результат
   (за потреби оновіть сторінку: Ctrl+R / Cmd+R)
```

## 8. Чого не варто робити

| Не робіть | Чому |
|-----------|------|
| Видаляти «Налаштування сайту» | Це єдиний запис налаштувань; видалення вимкнене |
| Міняти **Код сторінки** (slug) у сторінках контенту | Поле заблоковане навмисно — привʼязка до URL |
| Додавати нові «Блоки головної» | Список фіксований; додавання вимкнене |
| Завантажувати дуже важкі фото (десятки МБ) | Сайт може довго вантажитися |
| Залишати порожні обовʼязкові поля і чекати дива | Без **Зберегти** нічого не зміниться на сайті |

---

## 9. Якщо щось «не зʼявилося» на сайті

1. Переконайтеся, що натиснули **Зберегти**.
2. Перевірте галочки **Активне** / **Показувати на головній** / **На головній**.
3. Оновіть сторінку сайту в браузері.
4. Для зображень — перевірте, що файл реально завантажився (є превʼю в адмінці).
5. Якщо редагували блок головної — перевірте, що саме той блок відкрили (банер ≠ форма ≠ превʼю послуг).

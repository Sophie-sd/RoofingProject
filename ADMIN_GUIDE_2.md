## 1. Сторінка «Послуги» (`/services/`)

Сторінка = **шапка** + **картки послуг** + блок **«Чому ми»** + форма кошторису.

### 1.1. Шапка сторінки

**Меню:** Сторінки → **Послуги (заголовок)**

```
┌─ Сторінка «Послуги» ─────────────────────────────────────┐
│  Код сторінки: services   (не змінюється)                │
│                                                          │
│  Підпис над заголовком                                   │
│  Заголовок                                               │
│  Короткий опис                                           │
│  Текст сторінки (якщо потрібен додатковий HTML-текст)    │
│  Зображення заголовку / URL                              │
└──────────────────────────────────────────────────────────┘
```

Це верхня частина сторінки (hero): фон, заголовок, короткий опис.

### 1.2. Картки послуг

**Меню:** Контент → **Послуги (картки)**

```
┌─ Послуги ────────────────────────────────────────────────┐
│  Заголовок │ Варіант │ Порядок │ На головній │ Активне  │
│  …         │ Акцент  │ 0       │ ☑           │ ☑        │
└──────────────────────────────────────────────────────────┘
```

**Додати послугу:** кнопка **Додати послугу** → заповніть поля → Зберегти.

| Поле | Пояснення |
|------|-----------|
| Заголовок / Опис | Назва й текст картки |
| Іконка (Material Symbol) | Код іконки Google, напр. `roofing`, `handyman` (краще не чіпати без потреби) |
| Варіант | **Акцент** або **Темний** — стиль картки |
| Особливості | Список пунктів — **кожен з нового рядка** |
| Порядок | Черговість |
| На головній | Чи показувати на головній |
| Активне | Чи показувати взагалі |

### 1.3. Блок «Чому ми»

**Меню:** Контент → **Чому ми**

Картки з номерами (01, 02…) на сторінці послуг.

| Поле | Пояснення |
|------|-----------|
| Номер | Напр. `01` |
| Заголовок / Опис | Текст картки |
| Особливості | Пункти списку (з нового рядка) |
| Порядок / Активне | Як у інших списках |

## 2. Сторінка «Про нас» (`/about/`)

Сторінка = **шапка** + **текст/фото з блоку «Про нас»** + **фічі** + форма.

| Частина сторінки | Де редагувати |
|------------------|---------------|
| Верхній банер (заголовок, фон) | Сторінки → **Про нас (заголовок)** |
| Основний текст і фото секції | Головна → Блоки головної → **Про нас (превʼю)** |
| Картки переваг (іконка + текст) | Контент → **Фічі «Про нас»** |

### 2.1. Шапка

**Сторінки → Про нас (заголовок)** — як у послуг: підпис, заголовок, короткий опис, зображення, текст сторінки (HTML).

Поле **Текст сторінки** — довгий текст на сторінці «Про нас» (редактор з форматуванням).

### 2.2. Фічі «Про нас»

**Контент → Фічі «Про нас»**

| Поле | Пояснення |
|------|-----------|
| Іконка | Код Material Symbol (напр. `verified`) |
| Заголовок / Опис | Текст картки |
| Порядок / Активне | Видимість і черговість |

## 3. Сторінка «Портфоліо» (`/portfolio/`)

| Частина | Де редагувати |
|---------|---------------|
| Шапка (заголовок, фон) | Сторінки → **Портфоліо (заголовок)** |
| Фото / роботи | Контент → **Портфоліо** |
| Превʼю на головній | У роботі поставте галочку **На головній** |
| Заголовок секції на головній | Блоки головної → **Портфоліо (превʼю)** |

### Додати / змінити роботу

**Контент → Портфоліо → Додати роботу портфоліо** (або клік по існуючій)

```
┌─ Робота портфоліо ───────────────────────────────────────┐
│  Назва …………………  (напр. Приватний будинок)               │
│  Місто (slug) ……  (латиницею: bila-tserkva, kyiv)        │
│  Alt …………………  опис фото для доступності                 │
│  Дата …………………  (вільний текст, напр. 2024)              │
│                                                          │
│  Зображення: файл · превʼю · URL                         │
│                                                          │
│  Порядок · ☑ На головній · ☑ Активне                     │
└──────────────────────────────────────────────────────────┘
```

| Поле | Важливо знати |
|------|----------------|
| **Місто (slug)** | Латиницею, без пробілів. Однакові slug групують фото в один проєкт / фільтр за містом |
| **На головній** | Показувати на головній сторінці |
| **Активне** | Приховати без видалення |

Фільтри міст на сайті будуються з цих slug автоматично.

## 6. Заявки на кошторис

**Меню:** Заявки → **Заявки на кошторис**  
Або картка **Нові заявки** на головній панелі.

```
┌─ Заявки на кошторис ─────────────────────────────────────┐
│  Дата │ Нас. пункт │ Телефон │ Тип робіт │ … │ Опрацьовано│
│  …    │ …          │ …       │ …         │ … │ ☐          │
└──────────────────────────────────────────────────────────┘
```

| Дія | Як |
|-----|----|
| Переглянути деталі | Клік по рядку |
| Позначити опрацьованою | Поставте **Опрацьовано** у списку або у формі → Зберегти |
| Знайти | Пошук зверху (телефон, населений пункт) |
| Відфільтрувати | Фільтри справа / збоку (опрацьовано, тип робіт, матеріал…) |

Поля заявки (телефон, площа тощо) **тільки для читання** — їх заповнив клієнт на сайті. Редагувати можна лише статус **Опрацьовано**.

## 7. Чат (розмови з сайту)

| Розділ | Навіщо |
|--------|--------|
| **Розмови в чаті** | Список сесій відвідувачів (код сесії, імʼя, активність) |

Основна робота з відповідями зазвичай іде через Telegram-бота. В адмінці зручно **переглядати історію**.
